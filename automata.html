How It Works — Theory

Thompson's Construction (RE → ε-NFA)

  • Each symbol a → two states with a-transition
  • Concatenation: chain two NFAs via ε
  • Union (|): new start/accept, ε to both branches
  • Kleene (*): ε loop back + ε bypass
  • Plus (+): same as * but no initial bypass
  • Optional (?): ε bypass only

Subset Construction (ε-NFA → DFA)

Each DFA state is a set of NFA states. Starting from ε-closure(q₀), for each input symbol compute ε-closure(move(S, a)) to get the next DFA state. Repeat until no new states. A DFA state is accepting if any NFA state in its set is accepting.

ε-Closure

The ε-closure of a state S is the set of all NFA states reachable from S using only ε (epsilon) transitions — including S itself. This is computed using BFS/DFS over ε-edges.

Simulation

On the NFA tab, simulation tracks a set of active states and applies ε-closure after each symbol. On the DFA tab, exactly one state is active at a time, making simulation deterministic.
